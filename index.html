Circuit Education Game

Educational puzzle game designed as part of a bachelor thesis to teach fundamental circuit concepts through interactive gameplay. The game translates basic electrical principles into gameplay mechanics, where players solve progressively complex circuit-based puzzles to advance through levels. Core design focuses on learning through interaction, requiring players to understand and apply circuit logic rather than memorizing concepts, with mechanics structured around experimentation, problem-solving, and immediate feedback. The project is both designed and developed independently, covering gameplay systems, level design, and implementation.

Digital D&D Character Sheet

Interactive desktop tool for managing Dungeons & Dragons character sheets, built in WPF as a modular, real-time editing system. The application replaces traditional static character sheets with a dynamic digital interface designed to reduce in-game friction. It combines character stats, inventory, and session notes into a unified workspace optimized for fast updates during gameplay. A key design focus was flexibility and extensibility, including class-specific UI behavior, multiple editing modes (safe bulk editing, modal item editing, and notes management), and support for homebrew content. The system is built using a modular architecture with independent WPF User Controls and event-driven updates, enabling scalable UI expansion and maintainability.

D&D campaign notes web viewer

"Web-based tool for managing and visualizing Dungeons & Dragons campaign notes, designed for multi-party use with role-based content visibility. The system allows structured world and session note organization with support for markdown rendering, enabling readable and maintainable campaign documentation.
